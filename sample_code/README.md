

## 코드 공개 방식

소스코드는 팀 프로젝트 원본 전체가 아니라,
제가 담당한 주행 제어 관련 핵심 코드 위주로 업로드 했으며,
GitHub 포트폴리오용으로 주석 보강 및 일부 정리만 진행했습니다.

포함된 코드
- `My_motor.h / My_motor.c`
  - DC Motor 제어
  - Encoder 기반 RPM 계산
  - PI 제어
- `main_drive.c`
  - TIM6 기반 10ms 제어 루프
  - 초음파 거리 측정 및 필터링
  - Bluetooth 명령 기반 주행 시작/정지 처리
  - 거리 조건 기반 주행 로직
